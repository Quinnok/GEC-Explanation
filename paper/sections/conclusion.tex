\section{Conclusion}

This draft formulates edit-level faithfulness evaluation for GEC explanations as a three-layer problem: edit correspondence, counterfactual edit simulatability, and rule/evidence grounding. Reverse edit reconstruction remains useful as a leakage-controlled correspondence diagnostic, but the main planned behavioral test is whether an explanation predicts how the same GEC model's edit changes under controlled counterfactual variants. The current implementation provides real model-produced edit data and pilot diagnostics, but the scientific claims remain pending until counterfactual reruns, stronger explanation candidates, hard negatives, baselines, and human validation are completed.
