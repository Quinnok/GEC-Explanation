\section{Limitations}

This pilot uses automatic template explanations, open-source generated candidates, and synthetic hard negatives. They are useful for testing leakage, reconstruction behavior, counterfactual behavior, and negative controls, but they are not human gold explanations and should not be interpreted as learner-facing explanation quality. CoEdIT is run on a smaller CPU sample, and JFLEG alignment currently uses one primary reference while retaining all references for later multi-reference adjudication. ERRANT edit boundaries and error types may differ from valid alternative corrections. The current L2 simulators are weak, so the paper should not claim a strong counterfactual predictor. Most importantly, double-human faithfulness labels are still required to test whether any automatic metric agrees with human judgments.
