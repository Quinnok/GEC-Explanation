\section{Limitations}

This pilot uses many automatic template explanations constructed from ERRANT edit fields. They are useful for testing leakage, reconstruction behavior, and negative controls, but they should not be interpreted as learner-facing explanation quality. The current human-adjudicated set is intentionally stress-test-oriented rather than a natural random sample of GEC explanations. The structured baseline is intentionally narrow: it detects explicit edit statements, so strong performance on explicit and raw-edit conditions does not imply that free-form explanations are faithfully understood. ERRANT edit boundaries and error types may differ from valid alternative corrections, and future work should expand the human evaluation to balanced natural explanations from stronger teacher and open-source generators.
