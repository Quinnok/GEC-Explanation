\section{Limitations}

This pilot uses automatic template explanations constructed from ERRANT edit fields plus open-source generated candidates. They are useful for testing leakage, reconstruction behavior, counterfactual behavior, and negative controls, but they are not human gold explanations and should not be interpreted as learner-facing explanation quality. The current benchmark is still a pilot: CoEdIT is run on a smaller CPU sample, and JFLEG alignment currently uses one primary reference while retaining all references for later multi-reference adjudication. The structured baseline is intentionally narrow: it detects explicit edit statements, so strong performance on explicit and raw-edit conditions does not imply that free-form explanations are faithfully understood. ERRANT edit boundaries and error types may differ from valid alternative corrections. Most importantly, future work still needs double-human faithfulness labels to test whether any automatic metric agrees with human judgments.
