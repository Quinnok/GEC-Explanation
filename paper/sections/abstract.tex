\begin{abstract}
Grammatical error correction (GEC) systems increasingly provide natural-language explanations for their corrections, but a fluent explanation may fail to describe the edit behavior of the model that produced the correction. We study edit-level explanation faithfulness for model-produced English GEC edits and separate three evaluation layers: edit correspondence, counterfactual edit simulatability, and rule/evidence grounding. The pilot benchmark contains 700 actual model edits from EXPECT and JFLEG, 12,754 automatic explanation/control instances, and 120 counterfactual variants labeled by rerunning the original GEC models. Reverse reconstruction is useful as a leakage diagnostic, but drops from 0.640 Macro-F1 to 0.439 when target leakage is masked. Current counterfactual simulators are weak, with the best automatic L2 baseline reaching 0.297 Macro-F1. A reranking application further shows that high automatic scores can select template explanations that simply copy edits. These results support the benchmark and diagnostic framing, while human faithfulness and helpfulness validation remain blocked until real double annotation is completed.
\end{abstract}
