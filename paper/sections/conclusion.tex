\section{Conclusion}

This paper formulates edit-level explanation evaluation for GEC as a decomposed stress test over edit alignment, edit validity, rule correctness, evidence grounding, and overall faithfulness. The benchmark uses real model-produced edits, ERRANT extraction, counterfactual reruns, leakage controls, hard negatives, reranking diagnostics, and human-adjudicated edit-explanation labels. The main lesson is cautionary: reconstruction and surface leakage scores can look strong because they identify edit text, yet they remain weak tests of rule and evidence quality. The current evidence therefore supports using reverse reconstruction as an edit-alignment diagnostic, not as a standalone faithfulness metric. A stronger future version should validate the same decomposition on balanced natural explanations generated by strong teacher models and extend the human evaluation beyond the current stress-test distribution.
