\begin{abstract}
Grammatical error correction (GEC) systems increasingly provide natural-language explanations for their corrections, but an explanation that is fluent may still fail to describe the edit that the system actually made. This paper studies edit-level explanation faithfulness for GEC. We formulate a diagnostic question: if an explanation faithfully corresponds to a model-produced edit, can the edit be reconstructed from the original sentence and the explanation? We propose Reverse Edit Reconstruction, which maps a source sentence and an explanation back to a structured edit containing the edit span, source text, target text, operation, and error type. The method is intended to evaluate edit faithfulness, not grammatical validity or learner helpfulness. Empirical results are \texttt{[RESULT-PENDING]}.
\end{abstract}
