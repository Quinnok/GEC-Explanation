\section{Main Results}

\input{../results/paper_assets/main_results_table}

The automatic pilot supports three conservative findings. First, edit correspondence remains strongly affected by leakage: reverse reconstruction reaches 0.640 Macro-F1, but target-masked reconstruction drops to 0.439 and leakage-adjusted reconstruction to 0.496. Second, rule/evidence verification is the strongest current automatic L1/L3 baseline at 0.767 Macro-F1, ahead of the NLI lexical proxy and TF-IDF similarity. Because these labels are automatically constructed, this result is evidence of benchmark separability rather than proof of semantic explanation faithfulness.

Third, L2 counterfactual edit simulatability is substantially harder. We build 120 counterfactual variants from 60 model-produced edits and label each variant by rerunning the original GEC model. The best current automatic L2 simulator, a trained explanation-type prior, reaches only 0.297 Macro-F1. This negative result is important: counterfactual edit behavior is not captured by simple explanation-type or leakage cues.

The reranking application shows a complementary risk. The combined reranker obtains 0.935 automatic pairwise accuracy over 880 candidates for 80 edits, but it selects template explanations and edit-copy explanations as top-1 for 100.0\% of edits. A local open-source FLAN-T5 judge covers all 880 candidates without paid APIs, but scores only 0.455 pairwise accuracy, below the random baseline of 0.509. Thus the current reranking experiment is best read as a reward-hacking diagnostic, not as a solved explanation-selection method.
